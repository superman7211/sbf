专利创造性三步法：深入浅出手册

——「赋青春」复审无效部权威解读，轻松掌握创造性判断

一句话理解创造性

创造性 = 你的发明真的比现有技术聪明，而不是显而易见的组合。

第一章：三步法是什么（超简单版）

1.1 三步法的比喻

把创造性判断比作"找不同"游戏：

步骤 游戏中的操作 专利中的含义
第一步 选一张最接近的参照图 选一篇最接近的现有技术
第二步 找出两幅图的区别在哪里 确定你的发明和它的差异点
第三步 判断这个区别是不是"高级操作" 看这个差异是否需要创造性劳动

1.2 三步法详解

第一步：找"最接近的现有技术"

就是找一篇跟你的发明最像的现有技术。

关键点：

  • 技术领域相同（比如都是手机，都是药品）
  • 要解决的问题差不多
  • 结构/方法最接近
⚠️ 常见误区：不是特征最多的就最接近！如果一篇文件特征很多但解决的是完全不同的问题，不能用它。

第二步：找"区别特征"和"实际问题"

把你的发明和最接近的现有技术对比，找出哪里不一样。

这个不一样的地方，叫"区别特征"。

然后问："发明真正解决的是什么问题？"

⚠️ 重要：不是你声称的问题，是根据区别特征客观推导出的问题。

举例：你的区别特征是"可折叠"，实际问题可能不是"实现折叠"，而是"如何缩小收纳体积"。

第三步：看是否"显而易见"

问本领域的技术人员："你看了最接近的现有技术后，有没有想到加这个区别特征？"

  • 想到了 → 显而易见 → 没创造性
  • 没想到 → 非显而易见 → 有创造性

第二章：灵魂原则——整体考量

2.1 什么是"整体考量"

一个故事说明整体考量：

法拉利和五菱宏光都有发动机、轮胎、方向盘。但把它们各自零件拆开打乱，再随机组装，绝不等于一辆好车。

整体考量 = 零件要组装在一起看，看整体效果

2.2 整体考量在每一步的体现

第一步选最接近现有技术时：

  • 不只看特征数量
  • 要看技术构思是否接近
  • 同样解决"省油"问题的两款车，即便零件不同，构思可能更接近

第二步定技术问题时：

  • 不是把区别特征拆开看
  • 要看整体方案解决了什么问题
  • 各个特征组合在一起产生了什么协同效果

第三步判断启示时：

  • 不是把各个区别特征分别看有没有启示
  • 要看整体技术方案的启示
  • 防止"特征A有启示+特征B有启示=整体有启示"的机械叠加

2.3 防止"跑偏"

跑偏类型 错误做法 正确做法
孤立看特征 单独看每个区别特征是否显而易见 整体看各特征组合关系
混淆技术问题 用申请人声称的问题代替 基于客观效果重新定义
事后诸葛亮 看到发明后倒推是否显而易见 从现有技术正向推导

第三章：本领域技术人员——一个虚拟的人

3.1 他是谁

"本领域技术人员"是专利法里假想的一个人：

  • 知识：他懂申请日之前全世界的这个领域知识（但不横跨所有领域）
  • 能力：他会做常规实验和设计，但不会发明
  • 特点：他是一个没有创造力的"工具人"

3.2 他在判断中起什么作用

他是创造性判断的标准线：

本领域技术人员能想到 → 显而易见 → 没创造性
本领域技术人员想不到 → 非显而易见 → 有创造性

3.3 关于"公知常识"

公知常识 = 本领域技术人员普遍知道的东西
（比如教科书、工具书里的内容）

⚠️ 注意：说一个特征是"公知常识"，必须有证据！不能空口说"这是常规选择"。

第四章：发明构思——技术的"灵魂"

4.1 什么是发明构思

一个比喻：

苹果和华为都做手机，但技术构思不同：

  • 苹果的构思：封闭生态+极致体验
  • 华为的构思：通信技术+拍照能力

同样的零件，不同的构思，产出完全不同。

发明构思 = 发明背后的整体技术思路

4.2 发明构思为什么重要

核心原则：发明构思不同，即使结构相似，也可能具备创造性！

典型案例（克氏针夹持装置）：

对比 涉案专利 最接近现有技术
结构 触头绕尖嘴钳四周360°旋转 触头仅在底座单侧旋转
构思 全方位旋转夹持 单侧限位旋转

→ 构思完全不同 → 没有改进动机 → 具备创造性 ✅

4.3 发明构思差异如何否定创造性

当两个方案的构思完全不同时：

  1. 本领域技术人员看到最接近现有技术后
  2. 不会想到要用另一个构思去改进
  3. 因为没有动机，两个构思是"平行"的
  4. → 不存在技术启示 → 有创造性

4.4 构思判断三问

问题 答案
两个方案解决的是同一个问题吗？ 不同问题=不同构思
两个方案的技术手段是同一思路吗？ 不同思路=不同构思
两个方案追求的技术效果是同方向吗？ 不同方向=不同构思

第五章：最小技术单元——权利要求的"积木"

5.1 什么是技术特征

权利要求由一个个技术特征组成，每个特征是实现一定功能的最小单元。

产品专利的特征：部件、结构、连接关系、材料
方法专利的特征：步骤、顺序、条件、参数

5.2 贡献特征 vs. 非贡献特征

类型 定义 对创造性的影响
贡献特征 对解决技术问题有实质作用 影响创造性判断
非贡献特征 纯外形、尺寸等与技术问题无关 不影响创造性

2025年新规则：非贡献特征不算在创造性评价里！

举例：改进快门的相机专利，新增"外壳颜色是黑色"、"屏幕大小是6寸"——这些特征与快门改进无关，不影响创造性。

5.3 单一性与技术特征

单一性 = 一件申请里的多个方案，是不是同一个"发明构思"

判断方法：

  1. 拆出各方案的技术特征
  2. 找相同/相应的特征
  3. 看这些特征是不是"特定技术特征"（有创造性贡献）
  4. 有 → 单一性成立；无 → 需分案

第六章：技术启示——第三步的核心

6.1 什么算"启示"

启示 = 现有技术教了你把区别特征用到最接近现有技术上。

启示的三种来源：

来源 说明 举例
最接近现有技术本身 同一篇文件的其他部分暗示了你 文件1说"可以加电机"，你就加了
其他对比文件 其他文件给了教导 文件2公开了你想用的电机结构
公知常识 教科书/工具书里的常规手段 电机是本领域都知道的

6.2 没有启示的情形

情形 说明
技术领域差太远 机械领域的文件和电子领域的文件，通常不互相启示
作用不同 某个特征在最接近文件里是散热用的，你的发明里是支撑用的 → 没有启示
相反教导 现有技术明确说"不要这样设计"
有技术障碍 本领域技术人员知道这样组合会有问题

6.3 技术启示判断三要素（2025新规则）

启示要同时满足三点才构成"有启示"：

  • ✅ 启示解决的是相同/类似的技术问题
  • ✅ 启示采用的是相同/等效的技术手段
  • ✅ 启示能达到相同/可预期的技术效果

缺一不可！

6.4 2025年「赋青春」最新警示：禁止"拼凑式评述"

「赋青春」2025年4月发布的文章《"三步法"中技术启示的判断》给出了最重要的警示：

绝对禁止：把多个对比文件的零散特征随意拼凑，声称整体方案"显而易见"。

正确做法：必须以"最接近现有技术"为锚点，考察其他现有技术是否有明确的教导或启示将其特征结合进去。

典型案例（网球捡球清理装置）：

对比文件 技术方案 技术手段
最接近（对比文件1） 人工推动的移动捡球机 动态移动
对比文件2 围栏上固定魔术贴 静态固定

→ 两者技术手段本质不同（动态vs静态），且结合会产生新矛盾 → 无技术启示 → 有创造性 ✅

第七章：2024-2025年重要变化

7.1 三步法的进化

版本 核心变化
旧版 特征数量多=更接近
2024年 技术问题关联性 > 特征数量
2025年 发明构思差异优先；非贡献特征不纳入判断

7.2 问题-手段-效果一致性（2025新规则）

创造性成立需要三个东西对得上：

你的发明实际解决的问题
       ↓
你用的技术手段（区别特征）
       ↓
你达到的技术效果

举例：

  • 声称解决的问题：提高电池续航
  • 区别特征：屏幕刷新率从60Hz改成120Hz
  • 实际效果：屏幕更流畅（但续航反而下降）

→ 问题≠效果 → 没有一致性 → 无创造性 ❌

7.3 从"特征比对"到"构思判断"

旧思维 新思维
你的特征和现有技术比，有几个不同 你的构思和现有技术比，是什么关系
特征不同=可能非显而易见 构思不同=通常非显而易见
组合特征数量越多越强 构思独特性越高越强

第八章：实务速查清单

8.1 判断创造性自查问题清单

第一步选最接近现有技术：

  • 技术领域是否相同/相近？
  • 解决的技术问题是否相关？
  • 是否为了凑数而强行选了一个不合适的？

第二步定区别特征和技术问题：

  • 区别特征找全了吗？
  • 技术问题是根据区别特征的客观效果定的吗？
  • 有没有把申请人声称的问题当成客观问题？

第三步判断启示：

  • 现有技术是否给出了解决同一问题的启示？
  • 启示是相同/等效的技术手段吗？
  • 本领域技术人员有没有动机结合？
  • 公知常识有没有证据支持？

8.2 答复审查意见的"话术"

情形 推荐话术
最接近现有技术选择不当 "对比文件X解决的是Y问题，与本专利解决的Z问题不同，不宜作为最接近现有技术"
技术问题确定有误 "实际解决的技术问题应当是……而非审查员认定的……，理由是……"
不存在技术启示 "发明构思与对比文件存在本质差异……本领域技术人员没有动机……"
公知常识缺乏证据 "审查员主张该特征为公知常识，但未提供证据证明……"

8.3 无效程序的攻守要点

攻方（无效请求人）：

  1. 选好最接近现有技术——选最能打的
  2. 论证技术启示——动机+手段+效果
  3. 公知常识要有证据
  4. 重点攻击发明构思——说明本领域技术人员有改进动机

守方（专利权人）：

  1. 论证发明构思差异——构思不同=无动机
  2. 强调区别特征的作用不同——同样特征不同作用=无启示
  3. 质疑公知常识的证据
  4. 考虑主动修改——缩限保护范围换取维持有效
